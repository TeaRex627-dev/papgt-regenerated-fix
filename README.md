This repository contains a regenerated 0.papgt file that restores compatibility for mods after the latest game update.

Issue:
The game update changed the structure of the vanilla 0.papgt file, causing older modded versions to break the game.

Fix:
This file was rebuilt by merging the mod’s pack entry (0036) into the updated vanilla 0.papgt.

Result:
Game launches successfully with the mod enabled.

Notes:

Always regenerate 0.papgt from the latest vanilla file after updates
Do not reuse old modded versions
